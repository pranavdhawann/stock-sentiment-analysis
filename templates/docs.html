Documentation

Learn how to use InfoEdge for intelligent stock sentiment analysis

Getting Started

1. Search for a Stock

Enter a stock symbol (e.g., AAPL, TSLA) or company name in the search box.

2. View Results

Get instant sentiment analysis with news articles, charts, and keyword insights.

3. Analyze Trends

Use the sentiment meter and trend charts to understand market mood.

4. Make Decisions

Combine sentiment data with your own research for informed trading decisions.

API Endpoints

Search Stocks
GET /api/search_stocks?q={query}

Search for stocks by symbol or company name. Returns matching stocks with symbols and names.

Analyze Sentiment
POST /api/analyze_sentiment
Body: {"symbol": "AAPL"}

Analyze sentiment for a specific stock. Returns news articles, sentiment scores, charts, and keyword analysis.

Health Check
GET /health

Check the health status of the application.

Features Explained

Live Sentiment Meter

A visual gauge showing the overall sentiment percentage for a stock based on recent news analysis.

News Feed

Recent news articles with sentiment tags (Positive, Negative, Neutral) and confidence scores.

Price & Sentiment Chart

Interactive chart showing stock price movement overlaid with sentiment trends over the last 30 days.

Keyword Cloud

Visual representation of common themes and keywords in recent news, color-coded by sentiment.

Data Sources & Methodology

News Sources

We analyze news from multiple financial sources to provide comprehensive sentiment analysis.

Sentiment Analysis

Advanced NLP algorithms analyze news headlines and content to determine sentiment scores.

Real-time Updates

Data is updated in real-time to provide the most current market sentiment.

Confidence Scoring

Each sentiment analysis includes a confidence score to help you assess reliability.

Best Practices

Important Guidelines
  • Use sentiment analysis as one factor in your decision-making process
  • Consider the confidence scores when interpreting results
  • Look for trends over time rather than single data points
  • Combine with technical and fundamental analysis
  • Always do your own research before making investment decisions
